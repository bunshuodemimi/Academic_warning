# Academic_warning
学业预警系统

##📖 项目概述

Academic Warning System 是基于 Spring Boot + MySQL 开发的高校学业管理系统，聚焦解决传统学业管理人工统计低效、风险预警滞后问题，实现学业风险自动化识别与精准干预。

##✨ 核心特性

###🚨 智能预警引擎

规则化判定 - 自定义挂科 / 缺勤阈值，自动触发不同等级预警

多维度校验 - 联动成绩、考勤、选课数据，无死角识别学业风险

分级推送 - 预警信息精准推送给学生

预警流转 - 支持预警解除、升级、管理

###👤 多角色权限体系

学生端 - 学业数据查询、预警信息查看、消息接收

教师端 - 成绩 / 考勤录入、课程学生管理、学业数据统计

辅导员端 - 所管学院预警监控、学生学业干预、数据分析

管理员端 - 账号 / 基础信息配置、预警规则管理、全校数据管控

###🛠️ 数据管理能力

自动化同步 - 对接教务系统，批量同步学生 / 课程 / 成绩数据

批量操作 - 支持成绩 / 考勤批量导入、预警信息批量处理

数据安全 - BCrypt 加密存储密码、JWT 身份认证、权限粒度控制

字符兼容 - 基于 utf8mb4 字符集，支持生僻字 / 特殊字符无乱码

###🛠️技术栈

后端框架：Spring Boot 4.0.1 + Spring Web MVC

数据存储：MySQL 8.0+

AI 集成：LLM API（支持主流大模型）：百度千帆

安全机制：JWT Token + BCrypt 加密

数据访问：Spring Data JPA + HikariCP 连接池
